## 常用主题变量

在 `useAppStyles` 中可用的常用 token：

```typescript
token.colorPrimary; // 主色
token.colorSuccess; // 成功色
token.colorWarning; // 警告色
token.colorError; // 错误色
token.colorBgContainer; // 容器背景
token.colorBgLayout; // 布局背景
token.colorText; // 文本颜色
token.colorBorder; // 边框颜色
token.borderRadius; // 圆角
token.padding; // 内边距
token.margin; // 外边距
token.fontSize; // 字体大小
token.fontFamily; // 字体族
```

## 调试技巧

1. **查看实际应用的类名**：
   - 打开浏览器 DevTools
   - 找到目标元素
   - 查看类名（通常是 `.ant-*` 开头）

2. **测试样式优先级**：

   ```css
   /* 在浏览器控制台测试 */
   document.querySelector('.ant-conversations-item').style.padding = '20px';
   ```

3. **使用 React DevTools**：
   - 查看组件的 props
   - 检查 styles 属性是否正确传递

## 注意事项

- ⚠️ 使用 `!important` 时要谨慎，可能影响其他样式
- ⚠️ 全局 CSS 会影响所有相同类名的元素
- ✅ 优先使用组件的 `styles` 属性
- ✅ 使用 `token` 保持与主题一致
- ✅ 使用 `antd-style` 的 `css` 函数获得类型提示
